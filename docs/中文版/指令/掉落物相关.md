# 掉落物相关
> []为必填，()为选填

## drop add (权限：ed.drop.edit)
> /ed drop add \[名字] \[概率:%]
* `[名字]` 末影龙对应的配置文件中的unique_name
* `[概率]` 生效概率

为指定末影龙添加掉落物

## drop edit (权限：ed.drop.edit)
> /ed drop edit

在gui中修改掉落物配置

**推荐直接去reward文件夹里修改对应的掉落物配置文件**

## drop clear (权限：ed.drop.edit)
> /ed drop clear \[名字]
* `[名字]` 末影龙对应的配置文件中的unique_name

为指定末影龙清空掉落物

## drop gui (权限：ed.drop.gui)
> /ed drop gui (名字)
* `(名字)` gui的名字，如果缺省则使用config.yml中的"main_gui"

打开指定的gui，可浏览掉落物配置等
